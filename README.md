# Joan Comic

Static GitHub Pages build for the Jeanne d'Arc comic viewer.

## GitHub Pages

Publish this repository from the root directory. The app entry point is `index.html`.

The viewer depends on:

- `app.js`
- `styles.css`
- `texts_fr.json`
- `texts_en.json`
- `assets/*.ttf`
- `scenes/*.png`

No build step is required.
